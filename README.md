# First

The primary purpose of this repository is learning git as a beginner.

[See Demo](https://www.google.com)

## Index

- Installation
- How to start
- Supported node.js version

### Installation

```shell
$ git clone {repo addr} && cd {repo name}
$ npm install
```

### How to start
`$ npm start`

### Supported node.js version

`^V11.0`
